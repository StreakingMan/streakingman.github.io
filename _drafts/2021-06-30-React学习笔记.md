### JSX和React元素

- onClick等需要写的是函数而不是执行语句

  ```react
  // 错误，组件每次渲染都是alert
  <button onClick={	alert('click')}>
      // something
  </button>
  
  // 正确
  <button onClick={() => alert('click')}>
      // something
  </button>
  ```
